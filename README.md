# Стереотренировки

Веб-тренажёр для профилактики и тренировки при косоглазии. Работает прямо в браузере, офлайн, без установки. Открыл ссылку — и поехали.

Режимы:
- **Анаглиф** — картинки в двух цветах под красно-синие очки, расходятся/сходятся (под сходящееся и расходящееся косоглазие).
- **Стереограммы** — «Магический глаз», фигура прячется в шуме, очки не нужны.
- **Слежение** — движущаяся цель, в том числе режим «вглубь» для конвергенции.
- **Релаксация** — мягкие узоры для отдыха глаз, с таймером и звуком в конце.

Языки интерфейса: русский, украинский, английский, испанский, немецкий.

## Как подготовить свои картинки

1. Установи Python (если ещё нет) и библиотеку Pillow:
   ```
   pip install Pillow
   ```
2. Положи свои PNG/JPG в любую папку.
3. Запусти скрипт, указав путь к этой папке:
   ```
   python prepare_images.py "C:\путь\к\твоим\картинкам"
   ```
   Скрипт сожмёт картинки до 1024 px по большей стороне, переименует в `img1.png`, `img2.png`… и создаст папку `images/` с файлом `manifest.json`.
4. Скопируй получившуюся папку `images/` (вместе с `manifest.json`) в репозиторий рядом с `index.html`, заменив пустую.

Картинки из `images/` загрузятся автоматически при открытии сайта.

## Как выложить на GitHub Pages (пошагово)

### Вариант через сайт GitHub (без программ)

1. Зайди на https://github.com и нажми **New repository** (зелёная кнопка).
2. Назови репозиторий, например `stereo-training`. Сделай его **Public**. Нажми **Create repository**.
3. На странице репозитория нажми **Add file → Upload files**.
4. Перетащи туда `index.html`, `prepare_images.py`, и папку `images` (с твоими картинками и `manifest.json`). Нажми **Commit changes**.
5. Открой **Settings → Pages** (слева в меню).
6. В разделе **Build and deployment**, в поле **Branch**, выбери `main` и папку `/ (root)`, нажми **Save**.
7. Подожди 1–2 минуты. Вверху страницы Pages появится ссылка вида:
   ```
   https://ТВОЙ-НИК.github.io/stereo-training/
   ```
   Это и есть твой сайт. Им можно делиться с кем угодно.

### Вариант через VS Code

1. Установи расширение **GitHub Pull Requests and Issues** или просто пользуйся встроенным Git.
2. Открой папку с проектом в VS Code.
3. Внизу слева нажми на иконку аккаунта → **Sign in to GitHub** (если не залогинен).
4. Вкладка **Source Control** (Ctrl+Shift+G) → **Publish to GitHub** → выбери **public**.
5. После публикации зайди в репозиторий на github.com и включи Pages так же, как в шагах 5–7 выше.

## Обновление картинок потом

Прогоняешь `prepare_images.py` на новой папке → получаешь свежую `images/` → заливаешь её в репозиторий вместо старой (через Upload files или `git push`). Сайт подхватит новые картинки сам.

---

## Важно

Это вспомогательный инструмент, а не замена лечения. Тип косоглазия, режим и нагрузку нужно согласовывать с офтальмологом — упражнения, полезные при одном типе, могут не подходить при другом.
